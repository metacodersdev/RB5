## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  docker-compose -f docker-compose.mysql.yaml -p reviewboard_mysql up -d
```
